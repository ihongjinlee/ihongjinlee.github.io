## 📌 [vercel.com] Deploy

- `https://{저장소 이름}.vercel.app`

# 2. release branch

| branch  | version   |
| ------- | --------- |
| main    | 개발 버전 |
| release | 배포 버전 |

## 📌 [Git] release branch 생성하고 push

```console
$ git checkout -b release
> Switched to a new branch 'release'
$ git push --set-upstream origin release
```

## 📌 [Git] main branch로 돌아오기

```console
$ git checkout main
```
