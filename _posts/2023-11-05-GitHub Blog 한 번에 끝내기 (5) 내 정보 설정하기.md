## \_config.yml 수정

```yml
# Change to your timezone › https://kevinnovak.github.io/Time-Zone-Picker
timezone: Asia/Seoul

title: hongjin # the main title

tagline: 배우고 나누는 이슈 해결러 # it will display as the sub-title

description: >- # used by seo meta and the atom feed
  기억보다 기록하여 배우고 깨닫고, 공유 합니다.

# Fill in the protocol & hostname for your site.
# e.g. 'https://username.github.io', note that it does not end with a '/'.
url: "https://ihongjinlee.github.io"

github:
  username: ihongjinlee # change to your github username

social:
  # Change to your full name.
  # It will be displayed as the default author of the posts and the copyright owner in the Footer
  name: hongjin
  email: ihongjinlee@gmail.com # change to your email address
  links:
    # The first element serves as the copyright owner's link
    # - https://twitter.com/username # change to your twitter homepage
    - https://github.com/ihongjinlee # change to your github homepage
    # Uncomment below to add more social links
    # - https://www.facebook.com/username
    # - https://www.linkedin.com/in/username
```
